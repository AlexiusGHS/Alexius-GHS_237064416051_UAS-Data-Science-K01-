# UAS Data Science - Campus Recruitment

Project ini bertujuan untuk membangun model klasifikasi dalam memprediksi status placement mahasiswa serta mengidentifikasi faktor yang mempengaruhi keberhasilan placement.

Model utama yang digunakan adalah Logistic Regression dan dilakukan evaluasi menggunakan Accuracy, Precision, Recall, dan F1 Score.

Hasil analisis menunjukkan bahwa nilai tes kemampuan kerja dan pengalaman kerja memiliki pengaruh signifikan terhadap keberhasilan placement.
